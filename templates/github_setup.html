Setup Instructions
Create GitHub Personal Access Token

Go to GitHub Settings → Personal Access Tokens and create a new token with the following permissions:

  • repo - Full control of private repositories
  • public_repo - Access to public repositories
  • user - Read user profile data
Configure Token

Set your GitHub token in one of these ways:

Environment Variable (Recommended):
Set GITHUB_TOKEN=your_token_here in your environment variables
Config File (Testing only):
Edit config.py and set GITHUB_TOKEN = "your_token_here"
Test Connection

Click the "Test Connection" button above to verify your setup. Once connected, you'll be able to upload reports directly to GitHub.

Security Notes
Important: Never commit your GitHub token to version control. Use environment variables for production deployments.
Privacy: Reports can be uploaded to private repositories if your token has the appropriate permissions.
